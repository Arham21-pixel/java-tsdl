Issue: "JavaFX SDK not found"
→ Download from https://gluonhq.com/products/javafx/
→ Extract to C:\javafx-sdk-21
→ Make sure lib/ folder is inside

Issue: "java: command not found"
→ Install Java: https://www.oracle.com/java/technologies/downloads/
→ Make sure Java 21+ is installed

Issue: Compilation fails
→ Check all .java files are in src/ and src/ui/
→ Verify JavaFX path is correct
→ Try running from PowerShell (not cmd.exe)

Issue: App won't start
→ Check that bin/ folder created during compilation
→ Try running build.ps1 manually first
→ Check JavaFX download is complete and extracted
