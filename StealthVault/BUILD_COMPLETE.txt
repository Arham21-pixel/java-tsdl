🚀 QUICK START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. DOWNLOAD JAVAFX SDK (One-time setup):
   • Visit: https://gluonhq.com/products/javafx/
   • Download: Windows SDK (x64)
   • Extract to: C:\javafx-sdk-21
   • Verify the lib folder exists inside

2. LAUNCH THE APP:
   Open PowerShell in the StealthVault folder and run:
   
   .\start.ps1

3. LOGIN:
   • Username: (any text, e.g., "demo")
   • Password: (any text, e.g., "password")
   • Click LOGIN

4. EXPLORE:
   • View 5 pre-loaded vault items
   • Click categories to filter
   • Click SHOW to view full password
   • Click COPY to copy password
   • Click + ADD PASSWORD to create new entry
   • Click LOGOUT to return to login

  ✓ ObservableList for dynamic data binding

Design Patterns:
  ✓ MVC-inspired with Scene objects
  ✓ Callback functions for navigation
  ✓ Consumer<T> for event callbacks
  ✓ Runnable for action handlers
